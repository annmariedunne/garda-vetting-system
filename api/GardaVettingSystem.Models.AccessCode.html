Class AccessCode

Namespace

Represents a time-limited access code generated by an applicant for a specific organisation.

Access codes allow organisations to retrieve an applicant's vetting data without the applicant needing to re-enter their information.

Codes are designed to be revocable and time-limited in line with GDPR principles of integrity and confidentiality.

Code

Gets or sets the generated access code value.

Codes are 12-character alphanumeric strings generated using RandomNumberGenerator for cryptographic security.

Characters I and O are excluded to avoid visual ambiguity with digits 1 and 0.

Always set server-side — never accepted from form input.

[Required]
[StringLength(20, ErrorMessage = "Code cannot exceed 20 characters")]
[Display(Name = "Access Code")]
public string Code { get; set; }
