Class Applicant

Represents an applicant's personal profile in the Garda Vetting Data Reuse System.

Each registered user can have one applicant profile, which stores personal information required for Garda vetting applications.

The profile can be shared with organisations via generated access codes, eliminating the need for repeated data entry.
