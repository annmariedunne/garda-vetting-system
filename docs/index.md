# Garda Vetting System

A secure web-based data reuse system for Garda vetting applications, developed as a Final Year Project at Atlantic Technological University.

## Overview
The Garda Vetting System allows Irish volunteers to store their personal vetting information once and share it with multiple organisations via a generated access code, reducing the administrative burden of repeated vetting applications.

## Key Features
- Volunteer registration and secure authentication
- Personal vetting profile — store and manage personal data in one place
- Address history — full address history required for Garda vetting
- Access code generation — cryptographically secure 12-character codes
- Organisation validation — organisations can validate codes without logging in
- PDF export — download vetting profile as a PDF document
- GDPR compliance — full right to view, edit and delete personal data

## Technology Used
- ASP.NET Core Razor Pages (.NET 10)
- Entity Framework Core with SQL Server LocalDB
- ASP.NET Identity
- NUnit
- GitHub Actions (CI/CD)
- CodeQL and SonarQube
- DocFX for documentation
